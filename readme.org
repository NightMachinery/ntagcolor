#+TITLE: ntagcolor

=ntagcolor= colorizes its input (=stdin=) according to my [[https://github.com/NightMachinary/.shells/blob/master/scripts/zsh/auto-load/others/tags.zsh][filename-based tagging system]], which is simply adding =..some tag..= to the end of the filename.

Color tags are declared in the Go color table. Existing historical colors are
preserved, and CSS/web color names from Emacs are also supported, e.g.
=aliceblue=, =rebeccapurple=, and =yellowgreen=. Runtime color lookup uses a
checked-in generated table with precomputed RGB values and ANSI prefixes. See
[[file:docs/colors-from-emacs.md][docs/colors-from-emacs.md]] for the Emacs source and regeneration notes.
* Installation
#+BEGIN_SRC
go get -u -v github.com/NightMachinary/ntagcolor
#+END_SRC

* Build

Plain =go build= works because the generated runtime color table is checked in.
For local development, use the Makefile so generated styles are refreshed before
building or testing:

#+begin_src bsh.dash :exports code
make build
make check
#+end_src

See [[file:docs/build.md][docs/build.md]] for build targets and generated-file notes.

* Usage

[[file:readme.org_imgs/20201023_194243_P1tLlL.png]]

#+begin_src bsh.dash :results verbatim :exports code :wrap example
echo rainbow\ ..test..red..orange..yellow..green..emerald..aqua..teal..disruptor..blue..purple..gray..black..white..txt | ntagcolor
#+end_src

#+RESULTS:
#+begin_example
rainbow ..test..red..orange..yellow..green..emerald..aqua..teal..disruptor..blue..purple..gray..black..white..txt
#+end_example

By default, =ntagcolor= block-buffers stdout for throughput and flushes at EOF.
Use =--line-buffered= when another pipeline stage needs each rendered line as
soon as it is read:

#+begin_src bsh.dash :exports code
fd --color never | ntagcolor --line-buffered
#+end_src

End-to-end comparison on an 8.85 MB generated tagged file:

#+begin_src bsh.dash :results verbatim :exports both :wrap example
hyperfine --warmup 3 --runs 10 \
  '/tmp/ntagcolor-old < /tmp/ntagcolor-large.txt > /dev/null' \
  '/tmp/ntagcolor-new < /tmp/ntagcolor-large.txt > /dev/null' \
  '/tmp/ntagcolor-new --line-buffered < /tmp/ntagcolor-large.txt > /dev/null'
#+end_src
